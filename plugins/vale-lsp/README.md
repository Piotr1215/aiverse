# Vale LSP Plugin

Provides Vale prose linting for Markdown and MDX files via the Vale Language Server.

## Requirements

- `vale-ls` installed and in PATH
- Vale configuration (`.vale.ini`) in project root

## Supported Extensions

- `.md` - Markdown
- `.mdx` - MDX (JSX in Markdown)

## Installation

```bash
# Install vale-ls
go install github.com/errata-ai/vale-ls@latest
```
